# Security Policy

## Zgłaszanie Podatności

Bezpieczeństwo aplikacji Health Monitor jest dla nas priorytetem. Jeśli odkryjesz lukę w zabezpieczeniach, prosimy o odpowiedzialne zgłoszenie.

### Jak zgłosić?

**NIE** twórz publicznego Issue dla podatności bezpieczeństwa.

Zamiast tego:

1. Wyślij email na: security@health-monitor.example.com (zastąp rzeczywistym adresem)
2. Opisz szczegółowo problem
3. Dołącz kroki do reprodukcji
4. Poczekaj na odpowiedź (odpowiemy w ciągu 48h)

### Zakres

Podatności dotyczące:

- Nieautoryzowanego dostępu do danych użytkowników
- Obejścia mechanizmów PIN
- SQL Injection
- Wycieku wrażliwych danych zdrowotnych
- Manipulacji zapisanymi pomiarami

### Proces

1. **Zgłoszenie** - otrzymamy Twoje zgłoszenie
2. **Potwierdzenie** - odpowiemy w ciągu 48h
3. **Ocena** - przeanalizujemy problem (2-7 dni)
4. **Naprawa** - opracujemy i przetestujemy poprawkę
5. **Publikacja** - po naprawie opublikujemy informację o podatności

### Wspierane Wersje

| Wersja | Wsparcie |
| ------ | -------- |
| 0.2.x  | ✅ Tak   |
| 0.1.x  | ❌ Nie   |

## Dobre Praktyki dla Użytkowników

- Używaj silnych PIN (min. 4 znaki, różne kombinacje)
- Regularnie twórz backupy danych
- Nie udostępniaj swojego PIN innym osobom
- Aktualizuj aplikację do najnowszej wersji
- Nie instaluj aplikacji z nieoficjalnych źródeł

## Podziękowania

Dziękujemy wszystkim, którzy odpowiedzialnie zgłaszają problemy bezpieczeństwa!
